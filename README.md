# Employee Management System

Complete HR management system with PHP and MySQL.

## Features
✅ User Authentication (Admin, Manager, Employee)
✅ Employee CRUD Operations
✅ Attendance Tracking
✅ Leave Management
✅ Payroll Processing

## Tech Stack
- PHP 7.4+, MySQL 5.7+, Bootstrap 5

## Setup
1. Import `database/complete_schema.sql`
2. Configure `config/database.php`
3. Deploy to htdocs
4. Login: admin / admin123
